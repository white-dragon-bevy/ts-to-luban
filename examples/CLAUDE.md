
## Luban 配置系统

### 核心命令
```bash
pnpm run config:build          # 生成代码+数据（TypeScript + JSON）
```

## 测试

### 运行测试
```bash
pnpm run test              # 运行所有测试（TestEZ 框架）
```

### 测试位置
- 单元测试：与源文件同目录，使用 `.spec.ts` 后缀
- 集成测试：`__tests__/integration/` 目录

## 注释风格
- 整体符合 jsdoc 注释风格
- 类/函数注释：核心职责说明, 50字以内（如 /** 投射物核心组件 - 存储运行时状态 */）
- 接口,属性,字段注释, 必要但简洁.
- 行内注释：移除 ===== 分隔符，按流程步骤进行简短.
- 示例代码：全部移除，注释聚焦于"做什么"而非"怎么做"
