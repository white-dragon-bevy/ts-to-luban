本 cli 工具可以通过 typescript 类或者接口生成 **鲁班配置定义**，为配置驱动开发提供良好支持。

## 价值

传统的开发流程是先编写 **鲁班定义**， 再根据定义生成 ts 配置接口，问题如下:
1. 编写xml繁琐，且需要掌握 鲁班语法
2. 鲁班定义必须保存在同一目录，无法模块化分发
3. 策划编写配置时，缺少提示
4. 必须为配置编写消费者代码

本项目
1. 直接通过代码生成鲁班定义
2. 提供 json schema，提升配置体验
3. ts class 可以包含方法， 由配置直接生成实例

## 特性

### 通过interface 生成鲁班 bean

- interface to bean
- class to bean
- 支持继承
- jsdoc 转换
- 装饰器（仅支持 class）


### 生成鲁班 table
- @LubanTable() 装饰器
- 标记的 class 或者 interface 作为bean
- 额外创建 table 定义

### 装饰器

- Id
- Table
- ObjectFactory
- Ref
- Range
- Nominal 名义类型

### 支持鲁班验证器
见泛型工具 
 
### 对象工厂
- 通过泛型工具 `ObjectFactory<T>` 包装指定类型
- 提供帮助方法，通过配置创建对象
